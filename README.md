# Zomato Dataset Analysis

## Objective

Analyze restaurant data to identify factors influencing ratings, customer engagement, and restaurant popularity.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Data Cleaning

* Checked missing values
* Removed duplicate records
* Converted ratings into numeric format

## Exploratory Data Analysis

* Restaurant category distribution
* Online order analysis
* Table booking analysis
* Rating distribution
* Cost vs Rating analysis
* Correlation heatmap

## Key Findings

1. Dining is the most common restaurant category.
2. Online ordering restaurants generally receive better ratings.
3. Table booking facilities are associated with higher ratings.
4. Popular restaurants attract significantly more votes.
5. Restaurant cost has only a weak relationship with ratings.
